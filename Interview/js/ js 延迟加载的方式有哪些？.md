###  js 延迟加载的方式有哪些？
```
我了解到的几种方法分别是：
第一种：常见的将js脚本放在文档的底部，让jS脚本在最后来加载执行
第二种：给js加defer属性，这个属性会让脚本的加载和文档解析同步进行，在文档解析完成后才会去执行脚本，一般来说多个defer脚本会按顺序执行
第三种：给js加async属性，同样这个属性会让脚本的加载和文档解析同步进行，但是它是加载完后立即执行，不会等待文档解析完毕
第四种：动态创建DOM标签，可以对文档的加载事件进行监听，比如（DOMContentLoaded、load事件等），当文档加载完成后再动态的创建 script 标签来引入 js 脚本
```
