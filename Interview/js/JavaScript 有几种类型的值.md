### JavaScript 有几种类型的值，区别是什么

```
js 可以分为两种类型的值，一种是基本数据类型，一种是引用数据类型。
基本数据类型有Undefined、Null、Boolean、Number、String、Symbol、BigInt
引用数据类型则是Object类型，比如map、set、array、function、date等
```

```
两种类型的区别是存储位置不同
基本数据类型直接存储在栈中，占据空间小，大小固定。
引用数据类型存储在堆中的对象，占据空间大，大小不固定；引用数据在栈中存储了指针，该指针指向堆中该实体的起始地址。当解释器寻找引用值时，会先检索七在栈中的地址，取得地址后从堆里获得实体
```
